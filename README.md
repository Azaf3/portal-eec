 # Portal EEC

O Portal EEC nasceu em 2020 com o objetivo de democratizar o acesso ao conhecimento e à cultura. Começamos com pequenos workshops presenciais e, ao longo dos anos, crescemos exponencialmente.

Hoje somos uma plataforma de eventos educacionais e culturais reconhecida, com milhares de participantes, centenas de eventos realizados e parcerias com instituições e empresas. Nossa comunidade reúne estudantes, profissionais, palestrantes e organizadores que compartilham a paixão por promover aprendizagem, cultura e troca de experiências — não apenas tecnologia.

## Tecnologias

- React 18
- Vite
- Tailwind CSS

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```
